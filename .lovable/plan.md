

# Jitu's Premium Portfolio App

A mobile-first, app-like portfolio for a Digital Marketer & Web Developer — designed to feel like a premium startup product, not a typical website.

---

## Design System
- **Dark mode default** with glassmorphism cards (backdrop-blur, semi-transparent backgrounds)
- Soft purple/blue gradient accents on a near-black background
- Inter or Poppins font, rounded corners (2xl), subtle shadows
- Framer Motion for all animations: page transitions, scroll reveals, hover effects, number counters
- Dark/light mode toggle

---

## Layout & Navigation
- **Sticky bottom tab bar** (mobile-app style) with 5 tabs: Home, Work, Services, Skills, Contact
- Glass-effect nav background with active tab glow/highlight animation
- Smooth animated page transitions between tabs
- Floating "Hire Me" CTA button on scroll

---

## Pages

### 1. Home
- Animated hero with avatar, name ("Jitu"), title, and tagline
- Two CTA buttons: **Hire Me** → Contact, **View Work** → Work tab
- Animated stats cards (projects completed, leads generated, clients served, revenue generated) with counting animation on scroll

### 2. Work (Portfolio)
- Filter chips: All, SEO, Paid Ads, Web Dev, Automation, Social Media
- Card grid with project image, title, result summary, and tags
- Click a card → opens a detailed modal with problem, solution, tools used, results/metrics, and before/after visuals
- Sample placeholder projects included

### 3. Services
- Icon-based service cards (Website Dev, SEO, Google/Meta Ads, Marketing Automation, Funnel Building, Content Strategy)
- Cards expand on tap to show details and a pricing CTA
- "Book a Call" button at bottom

### 4. Skills
- **Marketing Skills** and **Tech Skills** sections with animated progress bars
- Tool/technology icon grid
- Experience timeline section with scroll-reveal animations

### 5. Contact
- Clean contact form (name, email, message) — frontend only, no backend
- Social buttons: WhatsApp, LinkedIn, GitHub
- Resume download button
- Calendly embed placeholder for meeting scheduling

---

## Extra Polish
- Loading skeleton states on all pages
- Scroll-reveal animations on cards and sections
- Smooth micro-interactions on buttons and cards (scale, glow)
- SEO meta tags
- PWA setup (installable from browser, works offline)
- Fully responsive: optimized for mobile first, scales beautifully to desktop
